## Installation
1. Pull doker image
```bash
docker pull manimcommunity/manim

docker run -it -p 8888:8888 manimcommunity/manim jupyter lab --ip=0.0.0.0
```

2. Use the Manim Setup.ipynb to install the required dependencies and setup the manim library.
